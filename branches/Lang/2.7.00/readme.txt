language readme:
The language items in this section are for translation and test purposes only. Many items have changed, many items are new and are in English. Please feel free to translate the English items and make suggestions to improve the existing translations. You should post your translations and suggestions to the forum at YaBBForum.com.

Yabb 2.7.00 is in UTF-8 only. Other encodings will cause the language files to fail.

To install a language pack for testing, you must upload the folder and files for the language into the Languages folder of your YaBB 2.7.00 forum. If there are Help files and folders, those should be uploaded into the Help folder of your YaBB forum.

There are currently 3 German versions - German (Formal), German_Formal, and German_Informal. German has a translated Help section, otherwise German and German_Formal differ only in the folder name in the Language folder..
