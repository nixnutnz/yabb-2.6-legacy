YaBB 3 is the next generation of YaBB bulliten board software.

We are adding even more features to YaBB in this version. While YaBB will continue to work with flat files as it does now, we are adding MySQL support as well as the capability to support almost any database or data source. We are taking all the HTML out of the board code and making YaBB completely skinnable to allow complete control over how your board looks. We will work to make YaBB play well with mod_perl and FastCGI, two modern technologies that allow Perl scripts to run at blazing speed.

For developers, YaBB 3 will be a major update to the codebase. We will bring new programming techniques to the codebase, such as objects where they make sense and strictures. We will be cleaning the code up and documenting it as much as possible.
